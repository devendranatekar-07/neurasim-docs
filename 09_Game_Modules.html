Three therapy variants share one ProjectScene ID. Each variant lives in its own module folder (Stereopsja, Fuzja, Konwergencja) with independent level and VFX setup. The scene selection on the tablet determines which variant is loaded.
CSaber ID 22 · CSaber
FolderCSaber (self-contained in THERAPY MODULES)
ControllerCSaberGameController
Params ctrlCSaberParametersController
Therapy typeVisual Acuity / Saber
Saber-slash game with Landolt C optotype targets for visual acuity testing. Extra params override or extend shared ones: object-distance-norm (shadowed as FloatParam defaulting to 6 m), vision-acuity (default 0.1), cube-move-speed (default 1). Unlike most games, CSaber assets are entirely self-contained under THERAPY MODULES/CSaber/.

3. Adding a New Game Module

The minimum required steps to wire a new therapy game into the framework:

  1. Add a new value to ProjectScene enum with the server-assigned integer ID (Assets/Shared/Scripts/ProjectScene.cs).
  2. Create a scene and add the three required GameObjects: UGCGameController subclass, UGCParametersController subclass (or base), UGCTherapeuticEventsController subclass (or base).
  3. Wire the scene into LoaderController.AsyncLoadScene() — add a case for the new ProjectScene value.
  4. Register the scene in Unity's Build Settings and add it to EditorBuildSettings.asset.
  5. Create a portal in the Hub scene (Assets/LOADER/NewLoaderGraphics/Hub.unity) and configure it to pass the correct game ID to the loader.
  6. Create a ComboManager subclass if the game needs difficulty progression; attach it to the scene.
  7. Add any game-specific parameters by subclassing UGCParametersController and declaring static CustomParam fields, or by creating ScriptableParameter assets and assigning them in the Inspector.
  8. Create the module folder under Assets/THERAPY MODULES/{GameName}/ for assets, and optionally under Assets/UGC-Protype/{GameName}/ for controller scripts.
The ProjectScene integer value must match the game ID on the dashboard server. Mismatches cause the loader to receive an unrecognised game ID and fall back to the Hub without loading the game.
